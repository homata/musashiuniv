**ダウンロード**
* [Google Colabファイルとデータ](./files/machine_learning_r0.zip)
    * **Googleドライブにアップロードする時の注意点**
        * ダウンロードされる「machine_learning_r0.zip」は圧縮ファイルです
        * 下記の手順でパソコン上で解凍してGoogleドライブにアップロードしてください
            1. パソコンにダウンロードした「machine_learning_r0.zip」ファイルは圧縮ファイルです。
            2. 「machine_learning_r0.zip」をパソコン上で解凍してください
                - Windowsの場合、右マウスメニュー「すべてを展開」を選択して解凍します
            3. 「machine_learning_r0.zip」が解凍されると「machine_learning_r0」というフォルダが作成されます
            4. 「machine_learning_r0」の下にある「machine_learning」フォルダーをGoogleドライブの「マイフォルダー」にアップロードしてください
